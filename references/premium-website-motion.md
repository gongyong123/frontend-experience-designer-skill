# Premium Website Motion

This file is retained for compatibility. For new work, prefer `marketing-site.md`.

Use premium motion for modern product websites, launch pages, portfolios, and immersive brand pages.

## Hero Reveal

- Start with a strong still composition.
- Reveal headline, supporting copy, and primary media in a controlled sequence.
- Use one dominant product/media movement: scale, rotate, mask reveal, light sweep, or depth shift.
- Keep text motion simpler than product motion.

## Scroll Storytelling

- Pin a product visual while text chapters change.
- Crossfade states instead of jumping between screenshots.
- Use scroll progress to reveal layers, labels, masks, or comparisons.
- Avoid long scroll hijacking unless the experience is explicitly cinematic.

## Product Choreography

- Hardware: slow rotation, exploded layers, material lighting, close-up details.
- Software: UI state transitions, feature panels, cursor-guided demos, command-to-result reveals.
- AI products: prompt-to-output transformation, branching possibilities, generated result reveal.
- Data products: chart build-up, connection lines, stream-to-insight transitions.

## Avoid

- Random floating blobs or orbs.
- Too many cards flying in.
- Animating every section the same way.
- Long delays before content is usable.
